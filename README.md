# CryptoTracker Pro

AI-powered cryptocurrency portfolio tracker and analytics platform.

## Features

- **Real-time portfolio tracking** - Monitor all your holdings in one place
- **AI predictions** - ML models predict market movements
- **Risk alerts** - Automatic notifications for price changes
- **Tax reporting** - Automated tax calculation for trading
- **API integration** - Connect with wallets and exchanges
- **Advanced charts** - Technical analysis tools

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

```bash
npm install -g vercel
vercel
```

Set environment variables:
- `STRIPE_SECRET_KEY`
- `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY`
- `COINGECKO_API_KEY` (optional)
